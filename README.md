# 🎬 FilmX – Movie Explorer Web App

FilmX is a web-based movie explorer application that allows users to search and discover movies using a public API. The application focuses on implementing JavaScript concepts such as API integration, array methods, and interactive UI design.

---

## 🎯 Purpose

The goal of this project is to build a functional web application that demonstrates:

* Fetching and displaying data from an external API
* Implementing search, filtering, and sorting features
* Creating a responsive and user-friendly interface

---

## 🌐 API Used

OMDb API
https://www.omdbapi.com/

---

## 🚀 Planned Features

* 🔍 Search movies by title
* 🎞️ Display movie details (poster, title, year, rating)
* 🎯 Filter movies by type (movie/series)
* 📊 Sort movies by year or title
* 📖 View detailed information for each movie

---

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind CSS)
* JavaScript

  * Fetch API
  * Array Methods (`map`, `filter`, `sort`)

## ⚙️ Setup Instructions

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Add your OMDb API key in `script.js`
