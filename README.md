# 🎬 FilmX – TV Shows Explorer Web App

FilmX is a modern web-based TV Shows explorer application that allows users to discover, search, filter, and sort shows using a public API.

---

## 🚀 Live Demo
👉 https://sakshamkejriwal.github.io/FilmX/

---

## 🎯 Features

- 🔍 Search shows by name
- 🎭 Filter shows by genre (Drama, Comedy, Action, Thriller)
- 📊 Sort shows by rating or name
- ⭐ Display ratings and show posters
- 🌗 Light/Dark theme toggle
- ⚡ Fast and responsive UI
- 🔄 Real-time updates using API data

---

## 🌐 API Used

- TVMaze API  
  https://api.tvmaze.com/shows

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Fetch API
- Array Methods (map, filter, sort)
